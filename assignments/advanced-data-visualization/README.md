# 📘 Assignment: Advanced Data Analysis & Visualization with Pandas

## 🎯 Objective

Practice advanced data analysis skills using `pandas` and `matplotlib`/`seaborn` to clean, explore, and visualize a dataset.

## 📝 Tasks

### 🛠️ Clean and explore a dataset

#### Description

Load a dataset into Pandas, perform data cleaning, and generate summary statistics that reveal important trends.

#### Requirements
Completed program should:

- Load the provided CSV dataset into a `pandas.DataFrame`.
- Identify and handle missing or invalid data.
- Create at least two new derived columns.
- Print summary statistics for key columns including mean, median, and counts.
- Save the cleaned dataset to a new CSV file.

### 🛠️ Visualize insights from the data

#### Description

Create charts that illustrate the dataset's most important trends and relationships.

#### Requirements
Completed program should:

- Generate at least two different visualizations (e.g. bar plot, line plot, scatter plot).
- Label axes, titles, and legends clearly.
- Save charts as image files (PNG or JPG).
- Optionally, interpret one chart by printing a short insight summary.
